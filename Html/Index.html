About WeatherWise

WeatherWise is a simple and clean weather application designed to provide you with the essential weather information you need without any clutter. Built with modern web technologies, it fetches live data from the OpenWeatherMap API to deliver accurate, real-time forecasts. Our goal is to make weather checking a quick, easy, and pleasant experience.
